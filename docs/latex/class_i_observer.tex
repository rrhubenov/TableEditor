\doxysubsection{Detailed Description}
Interface that describes an observer class. 

Used for the Observer pattern.

Whenever an event triggers on a Subject, \mbox{\hyperlink{class_i_observer_a3cb03fdce302c975435260441bceb5c1}{update()}} is called and the changes are received by the observers.
