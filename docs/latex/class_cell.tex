\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_cell}{Cell}} interface that describes what all cell classes must implement. 

A cell is an element that holds a value and knows how that value should be printer, calculated with, and parsed.

Using this interface all cell classes can execute operations on eachother.
